Principal Component Analysis

Date Mon 27 July 2015

This is the beginning of a series of me trying to explain Stat. / Machine-learning techniques / concepts in a succinct fashion.

What it is

It is a dimensionality reduction technique. It achieves data compression by picking out the most dominant features.

Type of ML technique

  • Unsupervised

Steps

  1. Perform eigenvalue decomposition of the mean-subtracted covariance matrix (\(d\times d\) where \(d\) is the number of dimensions)
  2. Sort eigenvectors (\(d\) of them) according to the descending order of the corresponding eigenvalues \(abs(\lambda_i)\) i
  3. Variance explained by each component is captured by the relative eigenvalue, i.e.
    $$abs(\lambda_i) / \sum_i abs(\lambda_i)$$
  4. Pick enough components to explain most of the variance and discard the rest of the components, hence reducing the dimension of the data

What it can achieve

  • pick up orthogonal (independent) linear components
  • should be numerically stable for most data unless there is serious
    collinearity between features such that the covariance matrix is ill-conditioned

Limitations

  • only works for picking out linear components but not nonlinear components
  • computationally intensive and scales as \(O(d^3)\)
  • have to recompute all data if there is new data
  • sensitive to the scaling of different dimensions
